Checkbox Selection in a TreePanel带有checkbox的树

This example shows simple checkbox selection in a tree. It is enabled on leaf nodes by simply setting checked:true/false at the node level.
树节点前带checkbox.在叶子节点上设置checked: false.则叶子节点前带上checkbox.如果设置checked: true.则checkbox会被选中.
当然也可以在folder上设置checked: false.比如To Do.

This example also shows loading an entire tree structure statically in one load call, rather than loading each node asynchronously.
在一次调用用一次性加载整棵树的结构. 而不是对每个节点异步加载.如果数据量大的话,应该异步加载,而不是一次性全部加载.

注释掉tree.getRootNode().expand(true),则树加载进来时不会展开所有节点.只有点击folder,才会加载folder下的leaf.
如果要和struts2等web框架结合起来.要返回给ext类似check-nodes.json的JSON格式数据.
