File Upload Field

This example utilizes a custom extension to implement a file upload field.实现文件上传的扩展方法 The js is not minified so it is readable. See file-input.js and FileUploadField.js.

Basic FileUpload最简单的文件上传
A typical file upload field with Ext style. Direct editing of the text field cannot be done in a consistent, cross-browser way, so it is always read-only in this implementation.上传文件的字段不可编辑

Basic FileUpload (Button-only)只有上传按钮
You can also render the file input as a button without the text field, with access to the field's value via the standard Ext.form.TextField interface or by handling the fileselected event (as in this example).

Form Example嵌入表单的上传组件
The FileUploadField can also be used in standard form layouts, with support for anchoring, validation (the field is required in this example), empty text, etc. This example also demonstrates using the buttonCfg option to provide a customized icon upload button.
FileUploadField可以被用在标准的表单布局中,支持字段非空验证等.上传按钮也支持自定义.
